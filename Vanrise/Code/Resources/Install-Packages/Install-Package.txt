Install-PackagesYou have to install below packages on new Web API project in order to be compatible with versions used in both projects Vanrise.Web and Vanrise.Web.Base

Open Package Manager Console under TOOLs > Library Package Manager

Select Default project : ProductX.Web

Execute below scripts

Install-Package Microsoft.AspNet.Web.Optimization -version 1.1.3
Install-Package Microsoft.AspNet.Mvc -version 5.0.0
Install-Package bootstrap -version 3.0.0
Install-Package jQuery -version 1.10.2
Install-Package Microsoft.AspNet.WebApi -version 5.0.0
Install-Package Microsoft.AspNet.WebApi.HelpPage -version 5.0.0
Install-Package RazorEngine -Version 3.9.3

Update attributes (Copy Local and Specific Version) to True in below DLL references:
System.Web.Http
System.Web.Http.WebHost
System.Web.Mvc
System.Web.WebPages
System.Web.WebPages.Deployment
System.Web.WebPages.Razor
System.Web.Razor

make sure that below DLLs has reference from C:\TFS\Vanrise\Code\Resources\DLLs\ path
Newtonsoft.Json.dll

Add reference to protobuf-net.dll

configure x64 on Runtime
