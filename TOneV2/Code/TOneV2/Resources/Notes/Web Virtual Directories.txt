Open cmd as Administrator 
Copy below script inside the command prompt.
This will create application pool, application, assign the application to pool, and add all virtual directories. You may have to change the site id and the bindings port.

cd c:\windows\system32\inetsrv
appcmd add apppool /name:"TOne.Web" /managedRuntimeVersion:v4.0 /managedPipelineMode:Integrated
appcmd add site /name:TOne.Web /id:10 /physicalPath:C:\TFS\TOneV2\Code\TOneV2\TOne.Web /bindings:http/*:8010:
appcmd set site /site.name:TOne.Web /[path='/'].applicationPool:TOne.Web
